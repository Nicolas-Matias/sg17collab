\section{Work programme}\label{work-programme}

SG\studyGroup{} reports use the place of a superscript next to a work item name to indicate its approval process

\begin{itemize}
    \item \textsuperscript{*} for TAP;
    \item \textsuperscript{**} for agreement; and
    \item no superscript is for AAP
\end{itemize}
